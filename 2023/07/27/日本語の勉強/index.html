日本語の勉強

发表于 2023-07-27 更新于 2023-10-02 分类于 日本語 阅读次数： Waline： 本文字数： 4.5k 阅读时长 ≈ 8 分钟

私の日本語のノート

新标日初级上

第八课

语法

  1. 之前で用于提示交通工具，现在拓展为“手段”以及“原材料”。 例句为:
  • 何==で==うどんを作りますか 乌冬面是什么做的
  • 李さんは日本語==で==手紙を書きます 小李用日语写信
  • 手紙を速達==で==送りました 用速递寄了信

2.　A把什么东西给B， 用に提示B，を提示物体，谓语部分是あげます（不能给我！） 例句为: - 私は小野さん==に==お土産==を==あげます 我给了小野先生一件礼物 - 小野さんは森さんにチョコレートをあげました 小野先生给了森先生巧克力

3.　A从B那里拿到什么东西，用に提示B，を提示物体，谓语部分是もらいます。除此之外，赠送者可以看成物品移动的起点，因此可以也可以用から（不能我给！） 例句为: - 私は小野さん==に==辞書==を==もらいました 我从小野哪里得到了字典 - 私は長島さん==から==写真==を==もらいました 我从长岛先生哪里得到了照片

4.　A见B，助词に提示B，谓语是会います 例句为: - 李さんは明日長島さん==に==会います 小李明天去见长岛 - 私は駅で森さん==に==会います 我在车站遇见了森先生 5.　もう表示“完了”“已经” 例句为: - 昼ご飯を食べました　（吃过午饭了吗）　ええ，==もう==食べました （是的，我已经吃过了）

6.　名词+（を）お願いします 例句为: - （打电话的时候） すみません、田中さんをお願いします （请田中先生接一下电话） - （在窗口请对方办手续） これ、お願いします （请帮我办一下这个）

  1. “别人给我”是くれます。该语法与あげます，もらいます构成“给”的全集。
  2. 「よ」结尾的回答带有一种隐含的意思：“对方此前是完全不知道的”；而「ね」回答的结尾隐含的意思是：“对方此前有一定的信息量”

表达及词语讲解

  • わかりました
  • 在别人给我的语法もらいます中，「に」/「から」都可以用，但对人一般「に」用；对组织机构一般用「から」
  • さっき和たった今都是过去式。たった今是比较近的时候的用法。さっき是稍微前一些时候的用法

课堂造句

  • 我从老师哪里得到了一个建议——先生からアドバイスをもらいました
  • 每个月从父母那里得到一千元的生活费——毎月両親から1000げんの生活費をもらいます
  • 这部电视剧我已经看过了——このドラマはもう見ました
  • 已经凌晨2点了啊，我要睡了——もう夜中の2時です、寝ます
  • 我已经不是小孩子了——私はもう子供ではありません

第九课

语法

  1. 日语中形容词分为两类，一类形容词以「い」结尾，做谓语时后面加上です；否定时把结尾的「い」变成「く」，です变成ないです或者ありません
  2. 需要特别注意的是，いいです的否定形式是よく　ないです
  3. 当时态是过去式的时候，肯定形式是将「い」变成「かった」；否定形式是将词尾变成く＋なかったです或者く＋ありませんでした
  4. 相对地，いいです的过去式以及否定形式是よかつたです和よく　ありませんでした
  5. 一类形容词可以直接修饰名词。但是要注意形容词和名词之间不能加の
  6. あまり+否定形式，表示程度不高。比如说这个汤不太热：「このスープはあまり熱くないです」
  7. 表示程度的副词：とても/たいへん（很，非常）> 少し/ちょっと(一点儿) > あまり(不太) > 全然(根本不) あまり/全然 后面要跟否定形式

表达及词语讲解

  • 多い和少ない不同于其他一类形容词，不能单独修饰名词，比如说很多书要用たくさんの本而不是多い本
  • あら是女性在感到惊讶的时候用的词语，由あれ变化而来
  • 「熱くないですか」否定疑问句，确认对方是否和自己持有相同看法。回答的时候可以用はい，也可以用いいえ
  • ちょうといいです表示刚刚好，正合适。、
  • お湯是“热水”或者“开水”的意思，水是凉水的意思。可以说熱いお湯/冷たい水，但不能说熱い水

第十课

语法

  1. 第二类形容词是不以「い」结尾的形容词，做谓语时后面加です。过去式是でした
  2. 二类形容词的否定形式是将です改成でわありません。过去式的否定是でわありませんでした。口语里でわ一般说じゃ。
  3. 二类形容词修饰名词的时候，是形容词+な+名词，比如“安静的城市”：静かな町
  4. 询问某人或者某事物的性质的时候用：どんな+名词。相当于汉语“什么样的”。何の与どんな的区别在与，何の是询问内容和材料，比如看什么书，煮什么菜；而どんな是询问性质，比如什么样的书（有趣），什么样的菜（辣）
  5. どうですか询问对方对某状态的意见或者感想，比如：この料理はどうですか；有时还用于劝诱对方进行某动作，比如：お茶，どうですか。
  6. 用于口语的连词：でも表转折，そして表并列。

表达及词语讲解

  • 漂亮的：「綺麗」，平假名是きれい，容易被错认为一类形容词。
  • あれえ表示感到吃惊或者意外而情不自禁发出的声音。
  • 是转换话题的时候使用，比如“（附和）。哎，请问这个美术馆里都有什么作品呢？”：ああ，そうですか。ところで，この美術館にはどんな作品がありますか。
  • 场所+中，表示场所的所有空间。时间+中，表示期间内的所有时间。比如“全世界”：世界中，“全年”：一年中。
  • 形容词+の，这里的「の」代指前文提到过的词语，可以避免重复使用词语。

第十一课

语法

  1. 表达A对B的情感：Aは　Bが＋　一类形容词/二类形容词です；「が」用于提示情感的对象。
  2. 分かります、できます、上手です、下手です、苦手です等有关能力的词语也可以用上述的句型；Aは　Bが＋　一类形容词/二类形容です；比如我会日语：私は日本語が分かります。
  3. A「や」B(など) 不完全列举，通常与「など」一起使用，作用于有点像，但是用于只列出许多项目中的两项，比如“吉田先生有时去中国，还有韩国什么的”：吉田さんは時々中国や韓国（など）へ行きます。
  4. 助词「から」用于陈述原因句子的末尾，可以先说明原因在陈述结论，也可以先陈述结论再说明原因。
  5. 连词「だから」独立使用在表示结果的句子前面，意思相当于“所以”，比较礼貌的说法是「ですから」。
  6. 表示频率的副词：「いつも」>「よく」>「時々」>「たまに」>「あまり」>「全然」.其中，「あまり」、「全然」后面要跟否定形式
  7. 「どうしてですか」相当于“为什么”。回答的时候在谓语后面加上「から」

表达及词语讲解

第十二课

语法

  1. Aは　B「より」+一类形/二类形です。意思是A比B更xxx

  2. A「より」B「の ほうが」+一类形/二类形です。意思是比起A，B更xxx。对比上一个语法可以发现，指示比较的基准（肯定）

  3. AはB「ほど」+一类形くないです/二类形ではありません。意思是A不如Bxxx。ほど指示比较的基准（否定）

  4. 「Aの中で」Bが　いちばん　一类形/二类形です。意思是在某个范围（A）内，B是最xxx的。比如说，在体育活动中，足球是最有意思的：スポーツの中でサッカーが一番おもしろいです。

  5. AとBと　どちらが　一类形/二类形ですか。意思是A和B哪一个更加xxx。「どちら」是二选一的疑问词，回答的时候用～の　ほうが...；如果两者程度相当，则回答どちらも～。例句： 　　　日本語と英語と　どちらが難しいですか。 　　　--日本語の　ほうが難しいです。 　　　コーヒーと紅茶と　どちらが好きですか。 　　　--どちらも好きです。

  6. 寻问三个以上的事物中，哪一个更xxx的时候，用：范围+合适的疑问词+が　いちばん　一类形/二类形ですか。比如，这些菜当中哪一个最好吃？この中で　どの料理が　一番　美味しいですか。

第二十课

语法

  1. Aは +动词基本形+ことが，表示能力。比如史密斯先生会弹钢琴：スミスさんはピアノを弾くことができます。
  2. 动词基本形+こと，可以将其变为名词词性，比如开车：車を運転すること\(\to\) 車の運転.
  3. 动词基本形+,...；表示一个动作在另一个动作之前发生。比如“来之前请打个电话”：こちらへ来る前に電話をかけてください。此外，前に也可以用于“开会前复印资料”类似的句子。
  4. 疑问词+か；表示“某时”“某地”“某人”的意思。比如“房间里有人吗？”：部屋に誰かいますか。
