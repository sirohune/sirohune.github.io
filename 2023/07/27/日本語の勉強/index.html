日本語の勉強

发表于 2023-07-27 更新于 2023-09-04 分类于 日本語 Waline： 本文字数： 3.2k 阅读时长 ≈ 6 分钟

私の日本語のノート

新标日初级上

第八课

语法

  1. 之前で用于提示交通工具，现在拓展为“手段”以及“原材料”。 例句为:
  • 何==で==うどんを作りますか 乌冬面是什么做的
  • 李さんは日本語==で==手紙を書きます 小李用日语写信
  • 手紙を速達==で==送りました 用速递寄了信

2.　A把什么东西给B， 用に提示B，を提示物体，谓语部分是あげます 例句为: - 私は小野さん==に==お土産==を==あげます 我给了小野先生一件礼物 - 小野さんは森さんにチョコレートをあげました 小野先生给了森先生巧克力

3.　A从B那里拿到什么东西，用に提示B，を提示物体，谓语部分是もらいます。除此之外，赠送者可以看成物品移动的起点，因此可以也可以用から 例句为: - 私は小野さん==に==辞書==を==もらいました 我从小野哪里得到了字典 - 私は長島さん==から==写真==を==もらいました 我从长岛先生哪里得到了照片

4.　A见B，助词に提示B，谓语是会います 例句为: - 李さんは明日長島さん==に==会います 小李明天去见长岛 - 私は駅で森さん==に==会います 我在车站遇见了森先生 5.　もう表示“完了”“已经” 例句为: - 昼ご飯を食べました　（吃过午饭了吗）　ええ，==もう==食べました （是的，我已经吃过了）

6.　名词+（を）お願いします 例句为: - （打电话的时候） すみません、田中さんをお願いします （请田中先生接一下电话） - （在窗口请对方办手续） これ、お願いします （请帮我办一下这个）

表达及词语讲解

  • わかりました
  • さっき和たった今都是过去式。たった今是比较近的时候的用法。さっき是稍微前一些时候的用法

第九课

语法

  1. 日语中形容词分为两类，一类形容词以「い」结尾，做谓语时后面加上です；否定时把结尾的「い」变成「く」，です变成ないです或者ありません
  2. 需要特别注意的是，いいです的否定形式是よく　ないです
  3. 当时态是过去式的时候，肯定形式是将「い」变成「かった」；否定形式是将词尾变成く＋なかったです或者く＋ありませんでした
  4. 相对地，いいです的过去式以及否定形式是よかつたです和よく　ありませんでした
  5. 一类形容词可以直接修饰名词。但是要注意形容词和名词之间不能加の
  6. あまり+否定形式，表示程度不高。比如说这个汤不太热：「このスープはあまり熱くないです」
  7. 表示程度的副词：とても/たいへん（很，非常）> 少し/ちょっと(一点儿) > あまり(不太) > 全然(根本不) あまり/全然 后面要跟否定形式

表达及词语讲解

  • 多い和少ない不同于其他一类形容词，不能单独修饰名词，比如说很多书要用たくさんの本而不是多い本
  • あら是女性在感到惊讶的时候用的词语，由あれ变化而来
  • 「熱くないですか」否定疑问句，确认对方是否和自己持有相同看法。回答的时候可以用はい，也可以用いいえ
  • ちょうといいです表示刚刚好，正合适。、
  • お湯是“热水”或者“开水”的意思，水是凉水的意思。可以说熱いお湯/冷たい水，但不能说熱い水

第十课

语法

  1. 第二类形容词是不以「い」结尾的形容词，做谓语时后面加です。过去式是でした
  2. 二类形容词的否定形式是将です改成でわありません。过去式的否定是でわありませんでした。口语里でわ一般说じゃ。
  3. 二类形容词修饰名词的时候，是形容词+な+名词，比如“安静的城市”：静かな町
  4. 询问某人或者某事物的性质的时候用：どんな+名词。相当于汉语“什么样的”。何の与どんな的区别在与，何の是询问内容和材料，比如看什么书，煮什么菜；而どんな是询问性质，比如什么样的书（有趣），什么样的菜（辣）
  5. どうですか询问对方对某状态的意见或者感想，比如：この料理はどうですか；有时还用于劝诱对方进行某动作，比如：お茶，どうですか。
  6. 用于口语的连词：でも表转折，そして表并列。

表达及词语讲解

  • 漂亮的：「綺麗」，平假名是きれい，容易被错认为一类形容词。
  • あれえ表示感到吃惊或者意外而情不自禁发出的声音。
  • 是转换话题的时候使用，比如“（附和）。哎，请问这个美术馆里都有什么作品呢？”：ああ，そうですか。ところで，この美術館にはどんな作品がありますか。
  • 场所+中，表示场所的所有空间。时间+中，表示期间内的所有时间。比如“全世界”：世界中，“全年”：一年中。
  • 形容词+の，这里的「の」代指前文提到过的词语，可以避免重复使用词语。

第十一课

语法

  1. 表达A对B的情感：Aは　Bが＋　一类形容词/二类形容词です；「が」用于提示情感的对象。
  2. 分かります、できます、上手です、下手です、苦手です等有关能力的词语也可以用上述的句型；Aは　Bが＋　一类形容词/二类形容です；比如我会日语：私は日本語が分かります。
  3. A「や」B，通常与「など」一起使用，作用于有点像，但是用于只列出许多项目中的两项，比如“吉田先生有时去中国，还有韩国什么的”：吉田さんは時々中国や韓国へ行きます。
  4. 「から」用于陈述原因句子的末尾，可以先说明原因在陈述结论，也可以先陈述结论再说明原因。
  5. 「だから」是“所以”的意思，是表示原因理由的连词，比较礼貌的说法是「ですから」。「だから」与「から」的区别是，「から」指示的不是完整句子，而「だから」独立使用在结果的句子前面。
  6. 用于口语的连词：でも表转折，そして表并列。

表达及词语讲解

  • 漂亮的：「綺麗」，平假名是きれい，容易被错认为一类形容词。
  • あれえ表示感到吃惊或者意外而情不自禁发出的声音。
  • 是转换话题的时候使用，比如“（附和）。哎，请问这个美术馆里都有什么作品呢？”：ああ，そうですか。ところで，この美術館にはどんな作品がありますか。
  • 场所+中，表示场所的所有空间。时间+中，表示期间内的所有时间。比如“全世界”：世界中，“全年”：一年中。
  • 形容词+の，这里的「の」代指前文提到过的词语，可以避免重复使用词语。
